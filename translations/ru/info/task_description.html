Напишите функцию, которая находит самую длинную палиндромную подстроку в строке. Ваш код должен быть как можно более эффективным!

Если вы нашли более одной подстроки, вы должны вернуть подстроку которая ближе к началу.

Входные данные: Текст в виде строки (string).

Выходные данные: Самая длинная палиндромная подстрока.

Предусловия: 1 < |text| ≤ 20
Текст содержит только ASCII символы.
